DrumGizmo v0.9.7 released

Updated: Sun, 2015-02-22 19:10
The Drumgizmo team have just announced the release of version 0.9.7 of Drumgizmo. The most notable change in this release is the implementation of realtime resampling. This means that Drumgizmo will match, in realtime, whatever samplerate you are working with, so you are no longer restricted to 44.1kHz! 
 
Check out the announcement at Linux Audio Users mailing list for full details.

JACK Aware

Updated: Thu, 2014-11-13 14:37
Any program that supports JACK is said to be JACK Aware. This means that it not only has support for JACK audio and MIDI ports, but that is also is capable of having it's timeline synced up via JACK. 
 
The combination of these two allow for modular set ups where programs can be interconnected, as well as being kept in sync. 
 
A list of JACK Aware programs can be found at jackaudio.org
 
  • Read more about JACK Aware
