JACK Aware

Updated: Thu, 2014-11-13 14:37
Any program that supports JACK is said to be JACK Aware. This means that it not only has support for JACK audio and MIDI ports, but that is also is capable of having it's timeline synced up via JACK. 
 
The combination of these two allow for modular set ups where programs can be interconnected, as well as being kept in sync. 
 
A list of JACK Aware programs can be found at jackaudio.org
 
  • Read more about JACK Aware

Ardroid (Android application): How to use it with Ardour

Updated: Thu, 2015-05-07 17:04


In this tutorial, I’ll go through how to use your Android device to control Ardour through your network.

Hi!
There’s a great Android application called Ardroid, which is intended to allow your Android device to work as a remote control for Ardour. Ardroid does this by sending OSC messages from your Android device, to you computer running Ardour. Ardour then lets your Android device control it through those messages.

MuseScore

MuseScore is a powerful music notation editor for the creation, editing and printing of various types of musical scores.

Some features include WYSIWYG creation and editing, support for unlimited staves, unlimited score length, up to four independent voices per staff, percussion notation, a plug-in architecture, and excellent looking notation.
